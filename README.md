## ▶️ Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```
Entre na pasta:
```
cd SEU-REPOSITORIO
```
Instale as dependências:
```
npm install
```
Execute o projeto:
```
npm run dev
```
